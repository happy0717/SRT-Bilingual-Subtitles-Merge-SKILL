# SRT 双语字幕合并工具

将英文和中文字幕合并为"上中文下英文"的双语字幕格式，通过**时间轴重叠算法**进行智能匹配，而非简单按序号拼接。

---

## 适配的字幕类型

### ✅ 完美适配

| 条件 | 说明 |
|------|------|
| **同源字幕** | 英文和中文字幕来自同一视频源（如 Netflix、Amazon Prime、Disney+ 等流媒体平台） |
| **时间戳高度一致** | 两种语言的时间轴基于同一母版制作，时间码差异在 ±200ms 以内 |
| **命名规则** | 文件名包含 `_轨道N_[eng]` 和 `_轨道M_[chi]` 后缀（如从 MKV/MP4 提取的字幕轨道） |

### ⚠️ 需调参适配

| 情况 | 建议 |
|------|------|
| **不同来源字幕** | 英文来自官方、中文来自字幕组（时间轴不完全一致），需降低 `--threshold` 到 0.2~0.3 |
| **帧率差异** | 如果两个字幕存在帧率偏差（24fps vs 23.976fps），脚本暂不自动处理，建议先用其他工具统一帧率 |
| **短字幕/歌词** | 大量 1~2 行的短文本，可适当提高阈值到 0.6 以提高匹配精度 |

### ❌ 不适用

| 情况 | 原因 |
|------|------|
| 时间轴完全不同 | 两个字幕非同一视频，时间码无对应关系 |
| 非 SRT 格式 | 目前仅支持 `.srt` 格式（ASS/SSA/VTT 暂不支持） |
| 三种及以上语言 | 当前设计为双语合并（上中下英），三语需修改脚本 |

## 使用方式

### 环境要求

- Python 3.6+
- 无需额外安装第三方库（仅使用标准库）

### 单文件合并

```bash
python scripts/merge_srt.py <英文srt> <中文srt> [输出srt]
```

示例：
```bash
python scripts/merge_srt.py "E01_轨道4_[eng].srt" "E01_轨道35_[chi].srt" "E01_bilingual.srt"
```

### 批量合并（推荐）

```bash
python scripts/merge_srt.py --batch <字幕目录>
```

脚本会自动扫描目录中所有 `_[eng].srt` / `_[chi].srt` 文件，按前缀自动配对，输出到 `merged/` 子目录。

示例：
```bash
python scripts/merge_srt.py --batch ./That_90s_Show_S01_sub
```

### 调整匹配阈值

```bash
python scripts/merge_srt.py <英文> <中文> output.srt --threshold 0.5
```

| 阈值 | 效果 |
|:----:|------|
| 0.2 | 宽松匹配，更多中文会被合并（适合时间轴偏差较大的字幕） |
| 0.4 | **默认值**，适合同源字幕 |
| 0.6 | 严格匹配，减少误配（适合条目密集的音乐/歌词字幕） |

## 合并策略

```
┌─────────────────────────────────────┐
│  英文SRT (基准)    中文SRT (匹配)    │
│       │                │            │
│  条目1 ──重叠>40%──→ 条目1 ✓        │
│  条目2 ──重叠>40%──→ 条目2 ✓        │
│  条目3 ──无匹配───→         (独立)  │
│  条目4 ──重叠>40%──→ 条目3 ✓        │
│       │                │            │
│  未匹配条目4 ──────────→ (独立)     │
└─────────────────────────────────────┘
```

1. **以英文时间轴为基准** — 英文通常分段更细，作为骨架
2. **时间重叠匹配** — 对每条英文，找到时间重叠 ≥40% 的中文条目
3. **中文在上，英文在下** — 输出格式为：序号 → 时间轴 → 中文 → 英文
4. **独立保留** — 未匹配的中文（标题卡、专属字幕）和英文（音效标注 `[music playing]`）各自保留

## Skill 信息

 `srt-bilingual-merge` Skill，可以被 AI Agent工具自动调用，实现字幕合并任务的自动化。

触发条件：当用户提到"合并字幕"、"双语字幕"、"中英文字幕合并"等关键词时，AI 会自动调用此 Skill 执行批量合并。
